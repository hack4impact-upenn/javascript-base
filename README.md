# Javascript Base

### Set-up
#### Clone the repository
```sh
git clone https://github.com/hack4impact/javascript-base.git
cd javascript-base
```

#### Install packages
```sh
npm install
```

#### Run the application
To run both the client and server:
```
npm start
```

To run just the client:
```
npm run client
```

To run just the serveR:
```
npm run server
```
